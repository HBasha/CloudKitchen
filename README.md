# Cloud Kitchen Menu

A simple, modern static website to display a cloud kitchen menu. Built with HTML, CSS, and JavaScript.

## Features
- Clean, responsive design
- Menu items rendered dynamically from JavaScript
- Easy to update menu

## How to Use
1. Open `index.html` in your browser to view the menu.
2. Edit `menu.js` to update menu items.

## Customization
- Update `style.css` for design changes.
- Add more menu items in `menu.js` as needed.
